2단계 캐릭터 만들기

🤖 2-1 AI에 요청하기

이미지 스타일과 이미지 크기(가로·세로 mm)를 정한 뒤 캐릭터 설정표를 생성합니다.

원하는 이미지 스타일
가로(mm)
세로(mm)
[2-1단계] - 원하는 이미지 스타일: <입력한 스타일> - 이미지 크기: <가로 ___mm × 세로 ___mm> - 1단계에서 생성한 줄거리: <1단계 결과 붙여넣기 내용> ★★위 줄거리를 기준으로 주요 인물을 추출하고, 각 인물별 캐릭터 참조 이미지용 프롬프트를 작성해줘. ※세부규칙: - 대상 연령에 맞는 이미지 스타일과 캐릭터를 설정할 것. - 각 인물은 고유한 캐릭터로 인식될 수 있도록 작성할 것. - 모든 캐릭터는 전신이 명확히 보이는 참조 이미지 기준으로 작성할 것. - 출력은 표 형식으로: 캐릭터 이름 / 역할 요약 / 캐릭터 참조 이미지용 프롬프트

3단계 페이지별 원고 만들기

🤖 3-1 AI에 요청하기

줄거리와 캐릭터 설정표를 바탕으로 페이지별 원고를 생성합니다.

생성할 원고 총 개수 (N)
한 페이지당 문장 개수(n)
[3-1단계] - 생성할 원고 총 개수 (N): <입력한 원고 개수> - 한페이지당 문장 개수(n): <입력한 문장 개수> - 1단계에서 생성한 줄거리: <1단계 결과 붙여넣기 내용> - 캐릭터 설정표: <2단계 캐릭터 설정표 내용> ★★위 줄거리와 캐릭터 설정표를 기준으로 원고 총 개수만큼의 원고를 페이지별로 나눠서 작성해줘. ※세부규칙: - 동화책에 들어갈 실제 글이므로 캐릭터 이름 앞에 외형 묘사를 반복하지 말 것. - 출력은 표 형식으로: 페이지 / 원고 - 이미지 생성 도구는 쓰지 말고 텍스트로만 답해줘.

📥 3-2 결과 붙여넣기

페이지별 원고 표를 붙여넣으세요. 페이지별 카드와 복사 버튼이 자동 생성됩니다.

4단계 장면묘사 + 최종 이미지 프롬프트

🤖 4-1 AI에 요청하기

캐릭터 설정표와 페이지별 원고를 바탕으로 장면묘사 프롬프트를 생성합니다.

[4-1단계] - 캐릭터 설정표: <2단계 캐릭터 설정표 내용> - 페이지별 원고: <3단계 페이지별 원고 내용> ★★위 페이지별 원고와 캐릭터 설정표를 기준으로, 배경이미지 프롬프트를 작성해줘. ※세부규칙: - 캐릭터가 등장할 때마다 캐릭터 설정표의 외형·질감·표정·체형·의상 묘사를 빠짐없이 포함할 것. - 각 페이지 원고에 어울리는 배경, 인물 배치, 감정, 분위기, 행동을 구체적으로 묘사할 것. - 문장의 끝은 자연스럽게 끝낼 것. - 프롬프트 끝에 반드시 “텍스트 없는 순수 일러스트(textless pure illustration, no text, no words)”를 포함할 것. - 출력 표: 페이지 또는 장면 / 장면묘사 프롬프트

📥 4-2 결과 붙여넣기 (= 최종 이미지 프롬프트)

장면묘사 표를 붙여넣으면 장면별 카드가 생성되고, 화풍·이미지 크기가 자동으로 함께 붙습니다.
